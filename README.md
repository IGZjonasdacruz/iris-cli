iris-cli
========

> The Iris command line interface.

Install this globally and you'll have access to the `iris` command anywhere on your system.

```shell
npm install -g iris-cli
```

## Usage

```bash
cd www
```

### Create New Project
```bash
iris init
```

### Reload iris.path
```bash
iris path
```

### Create Screen
```bash
iris screen
```

### Create UI
```bash
iris ui
```

### Create Resource
```bash
iris resource
```
